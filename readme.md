# Medisetu – Patient Portal

A lightweight and responsive Patient Portal web application that allows patients to log in and view their medical reports.

**Live Demo:** [https://medisetu-patient-portal.vercel.app/](https://medisetu-patient-portal.vercel.app/)  
**GitHub Repository:** [https://github.com/DHWANI54/medisetu-patient-portal](https://github.com/DHWANI54/medisetu-patient-portal)

---

## Features

- Patient Login Page
- View Individual Medical Reports
- Reports Listing Page
- Responsive and user-friendly UI
- Fast static deployment using Vercel

---

## Tech Stack

- **HTML5** – Structure and layout
- **CSS3** – Styling and responsiveness
- **JavaScript (Vanilla)** – Client-side functionality
- **Vercel** – Hosting and deployment
